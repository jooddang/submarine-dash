# Deep Dive Dash

## 한 줄 요약

브라우저 기반 횡스크롤 무한 러너 잠수함 게임 — 점프, 아이템, 산소 관리로 최고 점수에 도전하는 캐주얼 아케이드.

## 해결하는 문제

- 가볍게 즐길 수 있는 브라우저 기반 캐주얼 게임에 대한 수요.
- 모바일/데스크톱 모두에서 설치 없이 바로 플레이 가능한 경쟁형 게임 부재.
- 기존 무한 러너 장르에 산소 관리·아이템 수집·구조 시스템 등 전략적 깊이를 더함.

## 타겟 사용자

- **주요 사용자**: 브라우저에서 짧은 시간(1~3분) 캐주얼 게임을 즐기는 유저 (모바일 + 데스크톱).
- **부차적 사용자**: 주간 리더보드 경쟁을 즐기는 코어 유저, 데일리 미션·스트릭으로 꾸준히 복귀하는 리텐션 유저.

## 핵심 기능 (MVP Scope)

1. **무한 러너 코어 루프** — 자동 전진, 점프(탭/홀드), 갭·유사·성게 회피.
2. **산소 시스템** — 30초 산소, 자동 감소, 산소팩으로 회복, 0이면 게임 오버.
3. **아이템 시스템** — 황새치(비행+무적), 거북이 등딱지(유사 구조), 튜브 조각(낙사 구조), 돌고래(이중 점프).
4. **주간 리더보드** — 상위 5명, 주간 초기화, 기록 조회.
5. **데일리 미션 & 스트릭** — 3개 일일 미션, 연속 달성 시 돌고래 보상.

## 핵심 기능 외 (Out of Scope)

- 실시간 멀티플레이어 / PvP (고스트 레이스 향후 검토).
- 실물 화폐 결제 (IAP는 설계만 완료, 구현 미착수).
- OAuth / 소셜 로그인 (현재 자체 계정만).
- 네이티브 모바일 앱.
- 고급 안티치트 시스템.

## 비즈니스 모델 / 성공 지표

- **수익 모델**: (향후) 골든 아이템 IAP ($1~$8), 코인 이코노미, 코스메틱 스킨.
- **KPI**:
  - DAU / WAU
  - D1 / D7 리텐션
  - 일일 미션 완료율
  - 평균 세션 길이 (런 횟수)
  - 리더보드 제출률 (로그인 유저 중)

## 주요 도메인 용어

| 용어 | 정의 | 코드에서의 표현 |
|------|------|----------------|
| 런 (Run) | 한 번의 게임 플레이 (시작→게임오버) | `run`, `run_end` |
| 산소 (Oxygen) | 시간 제한 자원, 0이면 사망 | `oxygen`, `OXYGEN` |
| 갭 (Gap) | 플랫폼 사이 빈 공간, 추락 위험 | `gap`, hole |
| 유사 (Quicksand) | 밟으면 가라앉는 함정 플랫폼 | `QUICKSAND`, `isTrapped` |
| 성게 (Urchin) | 접촉 시 즉사하는 적 | `URCHIN` |
| 황새치 (Swordfish) | 비행+무적+3배속 파워업 | `SWORDFISH` |
| 거북이 등딱지 (Turtle Shell) | 유사 함정 구조 아이템 | `TURTLE_SHELL`, `RescueState` |
| 튜브 (Tube) | 4조각 수집→낙사 구조 충전 | `TUBE_PIECE`, `tubeCharges` |
| 돌고래 (Dolphin) | 이중 점프 소모 아이템 | `dolphin`, `dolphins` |
| 스트릭 (Streak) | 데일리 미션 연속 달성 일수 | `streak` |
| 코인 (Coin) | (미구현) 소프트 커런시 | `coin` |
| 스킨 (Skin) | (미구현) 잠수함 코스메틱 | `skin` |

## 제약 조건

- 게임 엔진 미사용 — 순수 HTML5 Canvas API 렌더링.
- 오디오 파일 미사용 — Web Audio API 합성 사운드만 사용.
- Redis 단일 DB — Upstash 무료 티어 제약 내 운영.
- Vercel 서버리스 — 콜드 스타트, 함수 실행 시간 제한.
